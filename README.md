# Telegram Bot Webhook Starter (Dotnet)

Production-ready skeleton for building Telegram bots using:

- .NET 10
- ASP.NET Core Minimal API
- PostgreSQL
- Background processing
- Docker
- Fly.io

This repository is intentionally minimal.

It demonstrates:

- Webhook-based bot architecture
- Long-running job processing
- Clean separation between API and Core logic
- Secure environment-based configuration
- Containerized deployment

This is not a framework.
This is not a full SaaS template.
This is a clean starting point for real production bots.
